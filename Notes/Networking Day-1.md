# Networking Day 1

## IP Adress

- Unique address assigned to a device on a network.
- Used so devices can communicate with each other.
- Typess:
  - IPv4
  - IPv6

## DNS (Domain Name System)

- Converts domain names into IP addresses
- Example:
  - google.com > IP address

## Router

- Connects different networks.
- Forwards data packets to the correct destination.

## HTTP

- Protocol used for communication between browser and server.

## HTTPS

- Secure version of HTTP.

## TCP/IP
- TCP:
  - Ensures reliable delivery of data.
- IP:
  - Handles addressing and routing.

## Packet

- Small unit of data transmitted over a network.
-Large files are broken into packets before transmission.
